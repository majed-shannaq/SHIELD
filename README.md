# SHIELD - Behavioral Fraud Detection

AI-powered fraud detection for banking apps using behavioral biometrics and Risk Scoring. Detects fraud without disrupting legitimate users.

**Team:** Majed, Al-Baraa, Rayhana  
**Hackathon:** Amad 2026 - Financial Regulations Track

## Results
- Precision: 100%
- Recall: 75%
- FPR: 0%
- AUC: 0.98

## Data
- **Real data:** [IEEE-CIS Fraud Detection](https://www.kaggle.com/c/ieee-fraud-detection) - 100,000 transactions
- **Simulated data:** Generated inside the notebook (50 users × 88 sessions)
